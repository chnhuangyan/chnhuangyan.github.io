WireShark in Ubuntu (Basic)

Install WireShark

  • Run the command in the terminal to install
    sudo apt-get install wireshark

  • Choose Yes when you see the following window while installing

  • Modify init.lua file

    • Run the command in the terminal
      sudo gedit /usr share/wireshark/init.lua

    • Change the last second line to
      –dofile(DATA_DIR..”console.lua”)

      Before modification:

      After modification:

  • Run the command to run WireShark
    sudo wireshark

Get Familiar with WireShark

  • Open any webpage and choose your network interface in WireShark

  • Then you can see the following panes

  • Protocol Filter
    Open www.google.com and capture http packets, click one and you can see all the fields of this TCP packet in the detail pane:

  • IP Filter
    ip.src ==192.168.1.102
    only display packets with source address 192.168.1.102

  • Port Filter
    tcp.port ==80
    only display TCP packets with port 80
    tcp.srcport == 80
    only display TCP packets with source port 80

  • Http Methods Filter
    http.request.method=="GET"
    only display packets with HTTP GET method

  • Logic Expression: and/or
    ip.src ==192.168.1.102 or ip.dst==192.168.1.102
    display packets with source address 192.168.1.102 or destination address 192.168.1.102

Statistics

  • Summary
    Click Statistics->Capture File Properties
    Then you can find the summary information
  • Protocol Hierarchy
    Click Statistics->Protocol Hierarchy
    Then you can see the information of Protocol Hierarchy
  • Conversations Click Statistics->Conversations
    Then you can see conversation information for all the protocols
  • Endpoints
    Click Statistics->Endpoints
    Then you can see the endpoints information for all the protocols
  • HTTP
    For HTTP communications, you can have the following statistics
  • IO Graphs
    Click Statistics->IO Graphs
    Then you can see the statistics in Graph. You can change filter for the statistics.
